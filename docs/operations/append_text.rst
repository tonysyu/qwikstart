===========
append_text
===========

.. include:: aliases.rst

Operation for appending text to a given file.

Required context
================

`file_path`
    |file_path description|

`text`
    Text that will be appended to `file_path`

Optional context
================

`prefix`
    default: `'\n'`

    Text added before appended `text`

`suffix`
    default: `''`

    Text added after appended `text`
